There should be 4 Java files,
The files are:
- rsvClient.java
- rsvServer.java
- Reservation.java
- Ticket.java

The rsvClient is responsible for handling all the client inputs to send requests over to the Server

The rsvServer is responsible for all the computing and calculating for any tasks

The Reservation is the method that is being using to communicate between the Client and Server

The Ticket class is to create objects that are required to fulfill the details of the prerequisites of a Airport Ticket
